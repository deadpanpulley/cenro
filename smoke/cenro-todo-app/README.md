# Todo App
This is a simple single-page todo web app built using plain HTML, CSS, and JavaScript. It includes features like adding, completing, deleting tasks, applying filters, localStorage persistence, keyboard-friendly controls, and responsive design.

## How to Use the App
1. Open `index.html` in your browser.
2. Add a new task by typing it into the input field and clicking 'Add Task'.
3. Toggle the checkbox next to a task to complete or uncomplete it.
4. Click the delete button next to a task to remove it from the list.
5. Use keyboard shortcuts `Enter` for adding tasks, `Delete` for removing tasks, and `Backspace` for editing tasks.

## Features
- **Add Task**: Adds new tasks to the list.
- **Complete/Uncomplete Tasks**: Toggles task completion status using a checkbox.
- **Delete Tasks**: Removes completed or uncompleted tasks from the list.
- **LocalStorage Persistence**: Stores tasks in localStorage so they persist across page reloads.
- **Keyboard-Friendly Controls**: Uses `Enter` to add, `Delete` for remove, and `Backspace` for edit tasks.
- **Responsive Design**: Automatically adjusts layout for different screen sizes.

## Additional Notes
- Tasks are saved in the browser's localStorage. Clearing the browser cache or history will clear all tasks.
- The app is designed to be simple and functional without additional libraries.
