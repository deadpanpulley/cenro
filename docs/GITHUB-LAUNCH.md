## README conversion essentials

1. Lead with a one-line promise and an authentic product screenshot/GIF.
2. Put a three-command Windows quickstart above any architecture discussion.
3. Explain what stays local, what needs consent, and what the app refuses to do.
4. Show the model kit and allow users to choose another Ollama tag.
5. Add a short architecture diagram and a contribution invitation.
