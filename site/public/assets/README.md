# Cenro launch imagery

Keep these stable production paths when adding the generated launch assets:

- `cenro-hero.png` — desktop product hero, wide landscape.
- `cenro-social-card.png` — 1280 × 640 GitHub/X social preview.
- `cenro-launch-cover.png` — 1080 × 1920 vertical launch-video cover.

`index.html` references the hero and the generated social card. Keep the Cenro mark as SVG in `../brand/`.
